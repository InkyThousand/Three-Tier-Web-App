**Terraform Files - What to Commit:**

- **`.terraform/`** ❌ **DON'T COMMIT** - Downloaded providers/modules (like node_modules)
- **`terraform.tfstate`** ❌ **DON'T COMMIT** - Contains sensitive data, use remote backend
- **`terraform.tfstate.backup`** ❌ **DON'T COMMIT** - Backup of state file

Your `.gitignore` should have:

*.tfstate
*.tfstate.*
